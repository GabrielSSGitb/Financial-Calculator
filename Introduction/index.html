HP-12C Financial Calculator

JavaFX Simulation Project

About the Project

This project simulates the legendary HP-12C financial calculator using Java and JavaFX. It replicates financial and mathematical operations including compound interest, present value (PV), future value (FV), payments (PMT), interest rate (i), and number of periods (n).

Button Functions

F / G → Access secondary functions
CLX → Clear display or memory (with F)
STO / RCL → Store and recall memory registers
CHS → Change sign (positive/negative)
1/x → Calculate inverse
√ → Square root
^ → Power (xʸ)
ln / eˣ → Logarithm and exponential functions
n! → Factorial (via G + 3)
+ → Addition
- → Subtraction
× → Multiplication
÷ → Division
ENTER → Push value to stack
XY → Swap X and Y values
PMT → Calculate payment amount
PV → Set present value
FV → Set future value
i → Set interest rate
n → Set number of periods

Main Methods Description (FXMLDocumentController)

  • btnF(): Activates F function prefix for secondary operations.
  • btnG(): Activates G function prefix for mathematical functions.
  • btnOn(): Clears all current data and resets the calculator.
  • CLX(): Clears the display or, when F is active, clears memory and registers.
  • Enter(): Pushes the current display number into the stack array.
  • stoBTN() / rclBTN(): Save or retrieve numbers from memory registers.
  • addUp(): Performs addition of stack numbers.
  • subtraction(): Performs subtraction between stacked numbers.
  • multiplication(): Performs multiplication of stack numbers.
  • division(): Performs division of stacked numbers.
  • nbtn(): Sets the number of periods or calculates it if zero.
  • ibtn(): Sets the interest rate or calculates it if zero.
  • pvbtn(): Sets present value or calculates it if zero.
  • fvbtn(): Sets future value or calculates it.
  • PMTbtn(): Sets payment or calculates it.
  • exponencial(): Calculates xʸ or √ depending on G state.
  • btn1ByX(): Calculates the inverse or eˣ depending on G state.
  • TotalPercentage(): Calculates ln or placeholder for %T function.
  • xy(): Swaps X and Y in the register stack.
  • dot(): Adds a decimal point to the input.
